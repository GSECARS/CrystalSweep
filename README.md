# CrystalSweep

![License](https://img.shields.io/badge/License-MIT-teal.svg) ![Python](https://img.shields.io/badge/Python-3.14-22558a.svg?logo=python&color=22558a)

CrystalSweep is beamline control and data acquisition software for single-crystal X-ray diffraction experiments. It provides an integrated interface for controlling synchrotron beamline hardware and collecting single-crystal X-ray diffraction data.

## Contributing
All contributions to the CrystalSweep project are welcome! Here are some ways you can help:
- Report a bug by opening an [issue](https://github.com/seescience/CrystalSweep/issues).
- Add new features, fix bugs or improve documentation by submitting a [pull request](https://github.com/seescience/CrystalSweep/pulls).

Please adhere to the [GitHub flow](https://docs.github.com/en/get-started/quickstart/github-flow) model when making your contributions! This means creating a new branch for each feature of bug fix, and submitting your changes as a pull request against the main branch. If you're not sure how to contribute, please open an issue and we'll be happy to help you out.

By contributing to the CrystalSweep project, you agree that your contributions will be licensed under the MIT License.
